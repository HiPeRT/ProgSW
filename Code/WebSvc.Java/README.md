# Simple Servlet

This is an example of a Jakarta Web Servlet in Java.

You can run some simple unit tests (see the unitTests package), or run the full server. To do this latter, you must install the Eclipse IDE for Web developers.

This has been tested with Tomcat 11.0.

It also works with Tomcat <10.0, but in this case you shall replace the jakarta.servlet-api library with the javax.servlet lib.
